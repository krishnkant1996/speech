Create react app using this command :
    npx create-react-app my-app
    cd my-app
    npm start

    Added speech-to-text npm libery for convert speech to text.
     npm i speech-to-text

    For references https://developer.mozilla.org/en-US/docs/Web/API/SpeechSynthesisUtterance
